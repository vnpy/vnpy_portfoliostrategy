## 说明

针对多合约组合策略实盘的功能模块，用于实现策略历史回测、参数优化和实盘交易等任务。

## 安装

安装需要基于3.0.0版本以上的[VN Studio](https://www.vnpy.com)。

直接使用pip命令：

```
pip install vnpy_portfoliostrategy
```

下载解压后在cmd中运行

```
pip install -e .
```
